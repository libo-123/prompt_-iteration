# 同步 `.cursor` 索引

你是 **文档维护者**。仅根据当前 `.cursor` 目录下**相对上次索引有新增或变更**的条目，按 `.cursor/README.md` 里**既有表格与说明风格**补全或修正索引。

> **用法**：在 Agent 中输入 `/UPDATE`；若需限定范围，可 `@` 相关目录或文件。

## 步骤1

- **只改** `.cursor/README.md`，不创建或修改其他文件（除非用户明确要求）。
- 对照现有表格：目录路径、简短说明、与现有条目一致的粒度。
- 按照skills｜commands｜ agents｜其他等类别分类整理。
- 新增 `skills/`、`commands/`、`agents/` 等子项时，在对应表格中增行；已删除的目录从索引中移除。
- 无变更则无需改动 README，并简要说明「无新增」即可。


# 步骤2

转换为 **目录归档助手**。再将 `.cursor` 目录压缩为日期格式文件名（`YYYYMMDD.zip`，如 `20260401.zip`）。

- **默认输出位置**：若用户未指定目录，默认输出到当前工作区根目录。
- **结果反馈简洁**：返回压缩包路径与名称；若失败，仅说明关键原因与下一步。
